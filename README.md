
# 🔐 Network Security Prediction System (ML + MLOps)

This project is an end-to-end Machine Learning system for detecting network security threats using a Kaggle dataset. It includes data ingestion, transformation, model training, experiment tracking with MLflow, and deployment using Flask.

## Features
- End-to-end ML pipeline
- Data ingestion and preprocessing
- Model training and evaluation
- Experiment tracking using MLflow
- REST API for predictions using Flask
- Modular and production-like structure

##  Workflow
1. Data Ingestion  
2. Data Validation & Transformation  
3. Model Training  
4. Model Evaluation  
5. Experiment Tracking (MLflow)  
6. Deployment (Flask API)

##  Tech Stack
- Python
- Pandas, NumPy, Scikit-learn
- MLflow (experiment tracking)
- Flask (API deployment)
- MongoDB


##  Project Structure
- networksecurity/ → Core ML pipeline
- notebook/ → EDA & experiments
- templates/ → UI 
- Artifacts/ → Saved models
- app.py → Flask app
- main.py → Training pipeline
